使用hexo借助github创建静态blog

背景介绍

What’s Hexo?

Hexo 是一个快速，简单并且强大的博客框架。你可以使用Markdown或者其他标记式语言编写网页。

安装Hexo

requirements

  • Node.js
  • Git

安装步骤

For windows

Install Git

点此下载Github for windows

For Windows Vista, 7, & 8

然后一步步安装就好了。

Install Node.js

点击此处下载Node.js

Install Hexo

命令行输入npm install -g hexo耐心等待完成。

For linux&Mac

Link

Notice for Macer

You may encounter some problems when compiling. Please install Xcode from App Store first. After Xcode is installed, open Xcode and go to Preferences -> Download -> Command Line Tools -> Install to install command line tools.

Install Node.js

The best way to install Node.js is installing with nvm.

cURL:

$ curl https://raw.github.com/creationix/nvm/master/install.sh | sh

Wget:

$ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh

Once installed, restart the terminal and run the following command to install Node.js.

$ nvm install 0.10

Or you can download the installer and install it.

install Hexo

Once all the requirements are installed, you can install Hexo with npm.

$ npm install -g hexo

使用

创建新文章

使用命令$ hexo new [layout] <title>

例如：$ hexo new use_hexo_create_blog会在\source\_posts\创建use_hexo_create_blog.md文件，注意，此处use_hexo_create_blog仅为文件名。

编辑文章

你可以用任何你喜欢的方式编辑.md文件。我用的是MarkdownPad 2
打开创建的文件：

title就是网页标题通常你会在浏览器标签栏看到它。

上传到Github

使用命令$ hexo generate生成档案。

使用命令$ hexo server本地预览(http://localhost:4000[port默认值为4000，可在 _config.yml中修改]）。

使用命令$ hexo deploy上传至Github。

以上命令均有简写形式$ hexo g``$ hexo s``$ hexo d。如果不进行预览，可以使用组合命令$ hexo d -g上传至Github。
